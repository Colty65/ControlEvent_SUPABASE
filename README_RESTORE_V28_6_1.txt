# ControlEvent - RESTORE v28.6.1 real desde v28.8

Este parche restaura los archivos de la v28.6.1 real, que fue la última versión confirmada como fluida.

Uso:
1. Descomprimir este ZIP.
2. Copiar su contenido encima del proyecto actual en GitHub/Desktop.
3. Commit + push.
4. Abrir producción en incógnito o borrar caché/PWA.

No es obligatorio borrar archivos antiguos para probar, porque el index restaurado apunta a los bundles v28.6.1.

Archivos antiguos opcionales que se pueden borrar más adelante están listados en el manifiesto.

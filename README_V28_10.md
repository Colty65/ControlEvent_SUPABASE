# ControlEvent v16_prod - Listas compactas móviles

Versión basada en v28.6.1 estable. Añade un modo compacto sólo para iPad/Android/equipos de baja memoria, para reducir el DOM de listas pesadas.

No toca INFOEVENTO, BACKUP, carga de datos ni guardado.

Consola:

```js
ControlEventMobileCompactLists.print()
ControlEventMobileCompactLists.enable()
ControlEventMobileCompactLists.disable()
ControlEventMobileCompactLists.compact('compras')
ControlEventMobileCompactLists.full('compras')
```

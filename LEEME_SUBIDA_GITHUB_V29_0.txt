Este paquete es para repositorios que ya tienen la v28.6.1.

Sube/arrastra estas carpetas y archivos respetando la estructura:

- package.json
- package-lock.json
- public/
- routes/
- server/
- README_V29_0.md
- README_V29_0_DELETE_OLD_FILES.txt
- PATCH_MANIFEST_V29_0.json

Después de subirlo, si GitHub conserva archivos antiguos, puedes borrar:

- public/app/legacy/legacy-bundle-before-modules-v28.6.1.js
- public/app/legacy/legacy-bundle-after-modules-v28.6.1.js

No es obligatorio borrarlos para que V29.0 funcione.

Si el móvil sigue cargando una versión antigua, abre la app y fuerza recarga o borra caché/datos del sitio. El service worker de v29.0 usa cache nueva: controlevent-shell-v50-24.

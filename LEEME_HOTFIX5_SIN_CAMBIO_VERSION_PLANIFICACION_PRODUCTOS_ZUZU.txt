Cambios incluidos:

1) PLANIFICACIÓN INICIAL - delimitación visual de productos
- Cada ficha/fila de producto queda delimitada con una línea negra suave, no muy gruesa.
- El objetivo es distinguir claramente dónde empieza y termina la información de cada producto aunque haya fondos de color por segmento/destino.

2) PLANIFICACIÓN INICIAL - agrupación inteligente de artículos equivalentes
- Se añade una normalización conservadora para evitar fichas separadas de artículos iguales escritos de forma distinta en catálogo/prompt.
- Casos cubiertos expresamente:
  - RON Barcelo / Ron BARCELÓ Añejo 0.7 L
  - WISKI JB / Whisky 5 Años J.B Botella 0.7 L
  - WISKI DYC / Whisky DYC 1L 40º
  - Johnnie/Jonie Walker
  - Brugal
  - Ginebras Larios, Beefeater/Beefetaer y Puerto de Indias
- La agrupación NO se aplica agresivamente a todo, para no juntar variantes reales como Coca-Cola normal, zero, 00, vinos distintos, etc.

3) Zuzu planificación - búsqueda/catálogo más flexible
- El backend también usa la misma idea de alias para localizar el producto correcto del catálogo cuando Zuzu lee donaciones/existencias o compras desde el prompt.
- Esto evita que Zuzu proponga dos líneas separadas si el prompt dice una forma corta y el catálogo tiene el producto con nombre más largo.

4) PRODUCTOS - precio de catálogo libre
- El mantenimiento normal de PRODUCTOS deja de hacer UPDATE directo cuando el producto ya existe.
- Ahora, si el nombre maestro no cambia, se actualiza precio/tienda/segmento/destino mediante la RPC ce_update_producto_catalogo_libre.
- Así se evita el bloqueo CE_BLOQUEO_FINALIZADO de ce_productos para cambios de catálogo.
- Las compras/donaciones históricas siguen intactas: el precio usado en un evento no se toca.

IMPORTANTE:
- Si la RPC ce_update_producto_catalogo_libre no existe en Supabase, la app mostrará un error claro indicando que hay que ejecutar ControlEvent_SQL_FIX_PRODUCTOS_CATALOGO_LIBRE.sql.
- Si ya se ejecutó ese SQL, no debería hacer falta repetirlo.

5) Ventana “Soy Zuzu, pregúntame lo que quieras...”
- Mientras Zuzu piensa, aparece una tarjeta con objeto animado tipo esfera/pulso.
- Se elimina el texto anterior informal y se sustituye por un mensaje claro: “Zuzu está pensando...”.

# ControlEvent v30.2

Versión basada en v30.1.

Corrección principal:
- El botón de menú **Mapa de productos** se abre ahora mediante manejadores directos `pointerdown`, `touchstart`, `mousedown` y `click`.
- Se elimina el tooltip/globo del botón para evitar que en PC/iPad intercepte o retrase la apertura.
- La pantalla sigue siendo una vista propia y estable, visible para GD, RW y RO.
- Se mantiene el rendimiento bueno de V29.4/V30.1.

No se ha modificado la lógica de INFOEVENTOS, BACKUP, Excel ni mantenimiento.

ControlEvent v11.1_prod - HOTFIX Ticket Auto auto-visible por CSS Gemini

Causa real encontrada:
- La nueva ventana Gemini libre introdujo reglas CSS genéricas para .ce-ai-overlay, .ce-ai-modal, .ce-ai-head, etc.
- Ticket Auto usa esas mismas clases.
- Como Gemini cargaba su CSS al arrancar, convertía .ce-ai-overlay en display:flex y hacía visible Alta asistida de COMPRAS sin que nadie la abriera.

Corrección:
- El CSS de Gemini queda acotado al id #ceGeminiLibreOverlay.
- Ticket Auto recupera su comportamiento: oculto salvo cuando tenga clase .open.
- No se cambia versión: sigue v11.1_prod.
- No se toca backend ni funcionalidad Gemini.

CARGA INICIAL DE DATOS

Importa PERSONAS, EVENTOS, TIENDAS, PRODUCTOS, INGRESOS, COMPRAS, DONACIONES y TICKETS desde una plantilla Excel.

La importación puede REPLACE o RESUME. Para relacionar hojas se usan códigos: PERSONA_CODIGO, EVENTO_CODIGO, TIENDA_CODIGO y PRODUCTO_CODIGO. El formato de carga y el de descarga de datos es el mismo. La hoja TICKETS admite IMAGEN_BASE64 o cruce por nombre de archivo.

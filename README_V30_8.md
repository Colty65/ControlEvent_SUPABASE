# ControlEvent V30.8

Versión funcional sobre V30.7.

- Mapa de productos renombrado a **Mapa de recursos**.
- Ordenación del mapa por **Tienda → estado/ticket → Producto**.
- Primero aparecen compras pendientes/otros gastos; después los **TKxx**.
- Las fichas ya compradas con TKxx se pintan en verde Excel transparente.
- Nuevo filtro desplegable múltiple por responsables **SOCIO**.
- El filtro afecta a compras y a donaciones con responsables seleccionados.
- Refuerzo específico del globo de **DONACION DE PRODUCTO** en iPhone/Android.
